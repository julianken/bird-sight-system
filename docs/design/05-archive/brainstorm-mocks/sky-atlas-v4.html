Spec-only changes (16 of 19 kinks) — encoded in design spec, no mock change
L1 K1 · Lede contract
4 lede templates evaluated by priority: zero-results, single-species, family-filter-active, default. Stale data drops the period clause. Defined for every (view, filters, freshness) tuple.
L1 K3 · Bottom-sheet contract
Drop <dialog>; use `
`. Snap heights: peek 96px, half 60% of content, full 100%-8px. Map gets `pointer-events: none` only at full snap.
L1 K5 · `` priority prop
Add `priority` boolean to <Photo>: masthead passes `priority={true}` → `loading="eager" fetchpriority="high"`. Feed thumbs default to `loading="lazy"`. Resolves LCP penalty on detail surface.
L1 K6 · Region as config
Extract region label to config: `frontend/src/config/region.ts` exports `REGION_LABEL`. Wordmark + lede source from same constant. Future expansion = one-line edit.
L1 K7 · Freshness label state machine
4 states: fresh / recent / stale / error. Read API exposes `meta.freshest_observation_at` via `MAX(inserted_at)`. Stale state escalates lede + meta line copy.
L2 K1 · Family-channel mechanism
JS-computed lookup table at `frontend/src/config/family-palette.ts`. `getFamilyChannel(code)` returns `{fill, on}` AA-paired. CSS custom-property path retired. Components consume via inline `style`.
L2 K2 · Cluster pill prop API
Pill takes `count`, computes tier internally. Thresholds (sand ≥100, ember ≥750) live in `frontend/src/config/cluster.ts`. MapLibre cluster layer imports the same constants.
L2 K3 · Token namespace migration
--accent → --color-decision-point (avoid notable collision). `--bg-*` / `--text-*` / `--border` get `--color-` prefix in production. Lint guard fails CI on `var(--accent)` outside mock dir.
L2 K4 · `` × `` boundary
They don't compose. `` owns 4-state machine: null-src → silhouette, loading → skeleton, loaded → img, error → silhouette. `` is page-level; never wraps ``.
L2 K5 · `` template
Single template: "Showing {filter terms} from the last {period}." Returns `null` at zero filters. Feed's "Sorted by recency" becomes separate ``. Period clause always present when component renders.
L3 K1 · Detail dialog heading + focus
Species name promoted to `

`. Dialog `aria-labelledby="detail-title"`. Initial focus targets heading, NOT close button. SR users hear species name on open.

L3 K2 · `` live-region
Always-mounted `role="status" aria-live="polite" aria-atomic="true"`. 500ms debounce on settled state. Filter-cleared transition holds "All filters cleared." for 1500ms before silence.
L3 K3 · Cluster pill ARIA
Pill renders `role="img" aria-label="{count} sightings"`. Tier (sky/sand/ember) is decorative density encoding — count is canonical. WCAG 1.4.1 satisfied by text count, not by color.
L3 K4 · Bottom-sheet role-switching
Sheet role flips with snap state: `region` at peek/half, `dialog aria-modal="true"` at full. `inert` set on map BEFORE role flips to dialog. ESC handler scoped to focus inside sheet.
L3 K5 · Reduced-motion policy
Single `frontend/src/styles/motion.css` rule: `transition-duration: 0ms !important; animation-duration: 0ms !important;` under `prefers-reduced-motion: reduce`. MapLibre `easeTo` reads matchMedia for `duration: 0`.
L3 K6 · Notable affordance spec entry
Documented constraint: "Notable affordance is card layout + label text. Color is amplification only. Future flat-row notable design must retain a non-color discriminator before removing card treatment."
Cross-cutting takeaway: Three of the spec fixes (family-palette, cluster-thresholds, filter-sentence constants) introduce a `frontend/src/config/` module. Combined with `region.ts` and `freshness-threshold.ts`, this becomes the auditable home for runtime parameters, taxonomies, and lookup tables that drive visual or behavioral branching. Token namespace migration (L2 K3) is the only fix that mutates the existing codebase — schedule first, before any v3 component work begins.
