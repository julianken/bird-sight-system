Sky Atlas v3 — agent ideas incorporated

Convergent agent ideas baked in. 9 mockups: map (desktop+mobile, light+dark), detail (desktop modal + mobile bottom-sheet, light+dark), loading state, feed lede.

Changes from v2 → v3 (in these mocks):
  1. Cluster pills — replace solid filled circles. White/dark fill + 2px density-coded stroke (Sky / Sand / Ember). Text contrast becomes arithmetic.
  2. Newspaper lede replaces "3,842 sightings" — "274 species seen across Arizona in the last 14 days."
  3. Filter-as-sentence — "Showing notable sightings from the last 14 days" — pairs with badge.
  4. Subtractive accent discipline — orange/cyan ONLY at decision points. Meta-label loses accent (now muted).
  5. Photo-as-anchor on detail — full-bleed photo with overlay title; species name at 36px on the photo.
  6. Bottom-sheet detail on mobile — 3 snap points (peek/half/full). Map remains live + pannable underneath.
  7. System skeletons — flat grey rectangles at content dimensions + 2px sunrise progress bar at top. No shimmer.
  8. System-font ramp (no webfont) — 6 sizes: 11/13/15/17/22/34. SF Pro / Segoe UI Variable / Roboto.
  9. Focus halo — visible on the active filter chip. 2px ring + 2px gap. Brand flourish, not OS default.
  10. Card-row top notable — first row of feed gets card treatment; rest stay flat.
